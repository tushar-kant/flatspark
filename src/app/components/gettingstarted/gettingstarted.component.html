Empowering Developers with Simplicity

Flatspark is a lightweight CSS library designed to streamline your web development process. It offers a comprehensive collection of well-maintained and customizable components, fostering efficient and consistent UI creation.

Your go-to CSS library for sleek, modern, and responsive web design

Using Flatspark

1. Import the CSS

npm install flatspark
import "flatspark/dist/index.css";
Or 
<link rel="stylesheet" 
href="node_modules/flatspark/dist/index.css"
>

Import the Flatspark CSS file in your main JavaScript file or directly in your HTML using a link tag.

2. Use Flatspark Components

Flatspark provides a variety of UI components like buttons, forms, and navigation. Refer to the documentation for detailed usage and customization options.
